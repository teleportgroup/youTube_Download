# YouTube Audio Downloader

A Windows desktop app for downloading YouTube videos as high-quality audio files.

![Windows](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## Features

- **Search YouTube** - Find songs by artist and title
- **Multiple formats** - MP3, FLAC, Opus, AAC, WAV
- **High quality** - Up to 320kbps for lossy, lossless FLAC support
- **Batch downloads** - Queue multiple URLs or entire playlists
- **Concurrent downloads** - Download up to 5 files simultaneously
- **Metadata embedding** - Artist, title, and album art embedded in files
- **Smart naming** - Files saved as `Artist - Title.ext`
- **Dark/Light themes** - Follows system preference or manual selection

## Installation

Download the latest installer from [Releases](../../releases) and run it. Everything is included - no additional software needed.

## Usage

1. **Search** - Type an artist and song name, click Search, then Add to queue
2. **Direct URL** - Paste a YouTube URL and click Download
3. **Playlists** - Paste a playlist URL with "Download playlists" enabled

Files are saved to your Music folder by default.
